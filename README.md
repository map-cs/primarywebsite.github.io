# MAP-CS Project Website

This repository hosts the public website for the **Mission Aligned Programs for Computer Science (MAP-CS)** project, an NSF-supported initiative focused on aligning computer science programs with institutional missions and student success goals.

## Project overview

The site provides:

- A high-level overview of the MAP-CS grant and its goals.
- Supporting materials and resources for institutions and collaborators.
- A process workbook and examples to assist with implementation.
- Project updates, contact information, and consulting details.

The site is built with **Jekyll** and a theme compatible with GitHub Pages, and is published at [**https://map-cs.org**](https://map-cs.org) via GitHub Pages.

## Repository structure (typical)

Some key files and directories you’ll see in this repo:

- `_config.yml` – Jekyll and theme configuration for the site.
- `index.md` – Landing page for the MAP-CS website.
- `welcome.md`, `overview.md`, `nsf-sigcse-acm.md`, `contact.md` – Main content pages.
- `supporting-material/` – Additional resources and supporting materials.
- `process-workbook/` – Workbook content, examples, and interactive tools.
- `other/` – Additional pages such as calendar, consulting, etc.
- `assets/` – Static assets (CSS, images, etc.), depending on theme.

Exact filenames and folders will vary slightly as the site evolves.
